# RAVN AI — Enterprise Security Platform

Python Flask application for website security monitoring, vulnerability scanning, and threat management.

## Quick Start

```bash
cd ravn-ai
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Open **http://localhost:5000** in your browser.

## Features

- **Landing page** — Professional corporate homepage
- **Dashboard** — Real-time security metrics with Chart.js visualizations
- **Websites** — Add, monitor, and scan web assets with live SSL/header/DNS checks
- **Alerts** — Threat notifications with acknowledge/resolve/escalate workflow
- **Reports** — Security and compliance report management
- **Settings** — Profile, notifications, security, API keys, and team management

## Tech Stack

- **Backend:** Python 3, Flask
- **Database:** SQLite
- **Frontend:** Jinja2 templates, custom CSS
- **Charts:** Chart.js
- **Scanner:** SSL/TLS validation, HTTP security headers, DNS SPF checks
